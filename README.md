## Others commands

> Change git editor

Use command:

```bash
$ git config --global core.editor "vim"
```

Or use:
```bash
$ git config --global core.editor "file path"
## Example
$ git config --global core.editor "F:/Program Files/Sublime Text 3/sublime_text.exe"
```
